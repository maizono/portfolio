My First iOS App

A project for my mobile development class, constructed with xcode and JSON.



My First iOS Application

Click the arrows to check it out.

Navigation Panel

This is the panel a user first sees when they open the app. I used a JSON data extracting method to retrieve the images and the text. Segues were used to connect each recipe to the next page.

Recipe Details

This screen, called a Detail View, showed a little bit more detail with the item: an enlarged picture, a small description, and the option to view even more information about it. These were also extracted from JSON data.

More Information

The Sub Detail View showed more in-depth information about the recipe: calories, servings, and preparation time. A link to the specific page of the recipe was also included. Once again, this information all came from JSON data.

External Website

The final part of the app displayed the AllRecipes site page for the chosen recipe. This shows after tapping the "Website" button on the previous screen. This URL was also extracted from JSON data.
