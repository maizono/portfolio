My First Website

About the Website

For IT 213, Introduction to Web Development, we had to create our own websites from scratch about a topic of our choosing. I decided to create a site for the fictitious "Kid Tech," a computer accessory and repair online market. The entire site was constructed with HTML and used an external CSS style sheet. You can view that style sheet here.
